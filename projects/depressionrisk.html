Overview

The Depression Risk Calculator is an AI tool for early identification of adolescents at risk of depression. It uses machine learning on non-invasive behavioral and environmental data to support timely intervention. This project focuses on translating research into a practical application for real-world prevention.

"Leveraging AI to proactively identify and support youth mental health through accessible, data-driven insights."

Core Idea & Approach

Mental health issues, particularly depression, are rising among adolescents. Early identification is key to effective intervention. This project aims to develop an AI model using readily available, non-clinical data (e.g., social media activity, sleep patterns, academic engagement proxies) to predict risk. The focus is on creating a tool that is both accurate and ethically sound, intended for guidance rather than diagnosis.

Technical Aspects

The system relies on a machine learning pipeline for risk prediction:

Data Ingestion

Secure collection of anonymized behavioral data.

Feature Engineering

Extracting relevant signals from raw data.

ML Model

Predictive modeling (e.g., Random Forest, Gradient Boosting).

Ethical AI

Bias detection and mitigation, explainability.
