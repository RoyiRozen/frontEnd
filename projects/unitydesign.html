Project Overview

Unity Design Tool is a VR application for Meta Quest that enables intuitive 3D shape creation and manipulation. Developed as part of Cornell Tech's CS 5650 course, this tool offers a powerful creative experience for designing in virtual space.

Key Features

Key object manipulation features include:

  • Object Creation & Selection: Create new shapes and select single or multiple objects
  • Transformation: Move, rotate, and scale objects individually or in groups
  • Customization: Change object shapes and colors with intuitive controls
  • Management: Duplicate and delete objects individually or in groups
  • Grab Move: Powerful feature to resize and reorient all objects in the scene
  • Help System: Built-in documentation to guide users through controls

Technical Implementation

Built in Unity for Meta Quest, this application features a universal helper system to prevent interaction conflicts and ensure smooth operation. Haptic feedback and custom controller visualization enhance the user experience with tactile responses and clear state indicators.

User Interaction

Intuitive Meta Quest controller mapping:

  • Right Trigger: Create new shapes
  • Right Thumbstick: Resize selection sphere
  • Right Bumper: Grab objects
  • A Button: Open shape menu (left) or color picker (right)
  • B Button: Delete selected objects (right) or open help (left)
