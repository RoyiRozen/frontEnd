Unity Design Tool

Development UI/UX Software XR Unity 3D

Project Overview

Unity Design Tool is a VR application for Meta Quest that enables intuitive 3D shape creation and manipulation. Developed as part of Cornell Tech's CS 5650 course, this tool offers a powerful creative experience for designing in virtual space.

Key Features

3D Object Manipulation

Intuitive controls for creating, scaling, rotating, and moving 3D objects in VR space with precision and ease.

Haptic Feedback

Realistic tactile responses for all interactions to enhance the sense of presence and provide better user feedback.

Material Editor

Customize object appearance with a comprehensive set of materials, colors, and texture options.

Technical Implementation

The application was built using cutting-edge technologies to deliver a seamless VR experience:

Unity 3D

Core development platform for VR application

C#

Scripting language for game logic and interactions

Oculus Integration

For VR headset compatibility and controls

ProBuilder

For in-editor level design and prototyping

Next Steps

We're continuously improving Unity Design Tool with new features and enhancements. Upcoming updates will include:

  • Multi-user collaboration for team-based design
  • Advanced material and lighting controls
  • Export/import functionality for 3D models
